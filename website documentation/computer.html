Computer.java

The calculations

Creates an instance of Predicate using the predicate(String) parameter of the constructor. This instance initializes and applies matchers (regex) in order to ensure that the predicate is well written (e.g. no missing variables after an operator) when it is finished, the computer instance applies the compute() method, this method : Owns and manages an expression stack and a list of tokens in postfix annotation Applies to each token the act() function. This function acts on the Expression stack, depending on the stack and the token it can be : an addition of variables, an operation on variables, the application of a function on variables etc... Calculates the execution time Displays the statistics (execution time, number of states of the automaton) Create an automaton if the last expression of the stack is of type Automaton which will then be drawn and written by the previous class.

Example of computation :

If I enter the command : "debug test "x+y=5";

Reading the token : x type : Variable
expression stack : [x]

Reading the token : y type : Variable
expression stack : [x, y]

Reading the token : +_msd_2 type : ArithmeticOperator
expression stack : [(x+y)]

Reading the token : 5 type : NumberLiteral
expression stack : [(x+y), 5]
computed ~:1 states - 25ms computed ~:2 states - 2ms

Reading the token : =_msd_2 type : RelationalOperator expression : [(x+y)=5]
