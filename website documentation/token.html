What is a token ?

The tokens are the elements that can be found in a formula of the b√ºchi arithmetic. We consider as tokens: letters of the alphabet, arithmetic operators, functions, left and right parentheses, logical operators, numbers, relational operators, variables and words. Token has two main functions: act() which is used to evaluate the Token, depending on the concrete class that extends it, and put() which is used to insert the token in the expression stack.

But the function act() act differently depending on the token.

How act() works on :

  • Alphabet Letter
  • Arithmetic Operator
  • Function
  • Left Parenthesis
  • Logical Operator
  • Number Literal
  • Relational Operator
  • Right Parenthesis
  • Variable
  • Word
