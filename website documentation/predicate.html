Predicate.java

Transformation of a formula into a predicate

Called by the Computer class, this class creates a predicate and initializes the matchers for the different Token. The function tokenize_and_compute_post_order() fills the expression stack by browsing the predicate string. When a Token type is identified, it is added to the expression stack by respecting the postfix notation.

Example

If I enter the command : "debug test "x+y=5";

postfix notation : [x, y, +_msd_2, 5, =_msd_2]
